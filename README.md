This project is a Vite + React app using Tailwind CSS and Framer Motion. It includes the full `RestaurantMenu` component.

## Quick start

1. Unzip and `cd` into the project folder.
2. Install dependencies:
   ```
   npm install
   ```
3. Run dev server:
   ```
   npm run dev
   ```
4. Build for production:
   ```
   npm run build
   ```
5. Preview production build locally:
   ```
   npm run preview
   ```

## Deploying

### Vercel
- Install Vercel CLI or use the Vercel web dashboard.
- Connect the repo and set framework preset to `Vite` (it auto-detects).
- No extra config is required.

### Netlify
- Create a new site from Git.
- Set build command: `npm run build`, publish directory: `dist`.
- (Optional) add `netlify.toml` for redirects or headers.
